Cloud Hosting using AWS EC2, VPC & MobaXterm

Secure & Scalable Website Hosting

This project demonstrates website deployment using AWS EC2 Ubuntu Server with Apache Web Server and remote access using MobaXterm.

Explore Project

AWS EC2

Virtual cloud server hosting the website.

VPC Security

Secure network isolation with controlled access.

MobaXterm

SSH-based remote server management and deployment.

Deployment Workflow

  • Launch Ubuntu EC2 Instance
  • Configure Security Groups
  • Connect via MobaXterm SSH
  • Install Apache Web Server
  • Upload Website Files
  • Host Website Publicly
